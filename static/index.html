Codphish - Malicious URL Detection

This project uses a machine learning model I built, trained on 16,800 URLs (half malicious, half benign), to predict if a website is malicious or not.

It looks at 13 different features of the URL and its WHOIS information to make its decision. The model has an accuracy of 98.45% when tested on 3,360 URLs, and you can find more about it here.
